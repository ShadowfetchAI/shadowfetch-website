Speed-first utility

Take a page. Save a PDF. Stay in the camera.

Fast PDF is a local iPhone and iPad utility built for paperwork velocity. In rapid mode, the camera stays live, each shot becomes its own PDF immediately, and the next page is ready without dropping back through a slow loop.

How it works

The app is optimized for real stacks of paper.

1. Choose a save path.

Use Files Each Time for one-offs, or create a dedicated Fast PDF folder for the fastest repeated workflow.

2. Start single or rapid capture.

Single mode opens the camera for one page. Rapid mode keeps the live camera session open between saves.

3. Each shot becomes its own PDF.

The app generates one PDF per image locally and writes it directly into the user-approved destination.
