Encrypt locally. Paste anywhere.

Hush is a lightweight iPhone and iPad utility that turns plain text into encrypted text on-device with a shared secret, then lets you paste the result into Messages, Mail, or any other app you already use.

What It Does

  • encrypt text locally with a shared secret phrase
  • decrypt locally on the receiving device with the same phrase
  • copy the encrypted output into any messaging app
  • avoid accounts, hosted inboxes, and message relay servers

Why It Exists

Hush is intentionally small. It does not try to replace a chat network. It protects the message text first, then gets out of the way.
