Common questions

What customers typically ask

Do I need an account?

No. Receipt to PDF does not require sign-in, does not create a Shadowfetch account, and does not upload receipts to a vendor-owned server.

Where do my PDFs go?

You can save through the native Files export sheet each time, or approve a parent location once so the app can create and reuse a Receipts folder.

Does the app use iCloud Drive?

Only if you choose an iCloud Drive location in Files. Sync is then handled by Apple’s Files infrastructure and your Apple account.

Why does the app ask for camera access?

The camera is used only for receipt scanning through Apple’s native document scanner. The capture flow requires a real iPhone or iPad camera.

What does the optional passcode do?

If you enable it, Receipt to PDF asks for the local 4-digit code when the app opens or returns from the background. The passcode is stored locally on that device.

Troubleshooting

First things to check

  • If the scanner doesn’t open, confirm camera permission is allowed for Receipt to PDF in iOS Settings.
  • If direct folder saving stops working, choose the Receipts folder location again so iOS can refresh the folder permission.
  • If you prefer not to keep a fixed folder, switch back to saving through Files each time.
  • If you export into iCloud Drive and don’t see the file on another device yet, wait for Apple Files sync to complete.
