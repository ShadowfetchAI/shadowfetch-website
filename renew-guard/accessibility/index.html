Renew Guard

Accessibility

Renew Guard is built with SwiftUI system controls, supports dark mode, and is intended to work cleanly with standard iOS accessibility settings.

Expected Accessibility Support

  • Dark Interface
  • Larger Text through system Dynamic Type behavior in standard SwiftUI controls

Before Claiming Additional Labels

If you plan to indicate support for VoiceOver, Voice Control, Differentiate Without Color Alone, Sufficient Contrast, or Reduced Motion in App Store Connect, test the app’s full common task flow first, including adding a subscription, editing one, importing a screenshot, viewing history, and using settings.
